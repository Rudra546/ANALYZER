# NumPy Analyzera

## 📌 Project Overview

This project is a **Menu-Driven NumPy Analyzer Program** written in
Python.  
It allows users to create **1D, 2D, and 3D NumPy arrays** and perform
various operations such as:

* Array Creation (1D, 2D, 3D)
* Indexing and Slicing
* Mathematical Operations
* Combine and Split Arrays
* Search, Sort and Filter
* Aggregate and Statistical Functions

This program is especially useful for **students learning NumPy** and
**Python array operations**.

\---

## 🚀 Features

### 1\. Array Creation

* Create 1D, 2D, or 3D NumPy arrays by entering values manually.
* Automatic reshaping based on user input.

### 2\. Indexing \& Slicing

* Access specific elements using indexing.
* Extract sub-arrays using slicing.

### 3\. Mathematical Operations

* Addition\\
* Subtraction\\
* Multiplication\\
* Division

### 4\. Combine \& Split Arrays

* Combine two arrays using concatenation.
* Split an array into multiple parts.

### 5\. Search, Sort \& Filter

* Search for a specific value in the array.
* Sort array in ascending or descending order.
* Filter values based on conditions.

### 6\. Aggregate \& Statistics

* Sum
* Mean
* Median
* Standard Deviation
* Variance
* Minimum \& Maximum
* Percentile Calculation
